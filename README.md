# node-chat

You can see this chat app working locally by firing up different/incognito browsers to act as multiple users.

## Installation
* `brew install redis`


## Tabs on tabs on tabs
* `grunt watch`
* `npm run start`
* `redis-server`


## Browser
* [localhost:3000](http://localhost:3000/)
